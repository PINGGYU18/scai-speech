\## 실습 보고서



\### 실습 1. 소리 만들기와 샘플링



1\. \*\*샘플링 레이트가 4000, 8000, 16000 Hz일 때 샘플 개수\*\*

&#x20;  - 각각 4000개, 8000개, 16000개. duration이 1초이므로 N = fs × T = fs 가 그대로 샘플 개수가 됨.

2\. \*\*수식값과 `len(x\_test)`가 같은가?\*\*

&#x20;  - 같음. `np.arange(int(sr\_test\*duration))`가 0\~fs×T개의 샘플을 만들기 때문에 N = fs × T 와 정확히 일치함.

3\. \*\*샘플링 레이트가 커지면 그래프의 점은?\*\*

&#x20;  - 더 촘촘해짐. 같은 파형을 한 주기당 더 많은 점(≈ fs/440)으로 찍게 됨.

4\. \*\*샘플링 레이트가 커지면 파일 크기는?\*\*

&#x20;  - 보통 커짐. 1초에 저장하는 숫자 개수가 늘어나 데이터양이 비례해서 증가함.



\### 실습 2. 정규화와 양자화



1\. \*\*정규화 전/후 최대 진폭\*\*

&#x20;  - 정규화 전: 3.5 (`big\_x = 3.5 \* x`), 정규화 후: 1.0

2\. \*\*`big\_x`와 `norm\_x`는 수식의 어디?\*\*

&#x20;  - `big\_x` → 원래 신호 x\[n] (분자), `norm\_x` → 정규화 결과 x\_norm\[n] (= big\_x를 최대 절댓값으로 나눈 값)

3\. \*\*가장 계단처럼 보이는 그래프는?\*\*

&#x20;  - 2 bit. 표현 단계 L = 2^(b-1)-1 이 가장 작아(L=1) 단계가 거칠어짐. (8 bit는 L=127)

4\. \*\*비트 수가 낮아질수록 소리는?\*\*

&#x20;  - 점점 거칠고 지직거리는(양자화 잡음 섞인) 소리로 들림.



\### 실습 3. FFT와 Spectrogram



1\. \*\*FFT 큰 봉우리 위치\*\*

&#x20;  - 440 Hz와 1000 Hz 근처 두 곳. 혼합 신호를 이 두 주파수로 만들었기 때문.

2\. \*\*`freqs` 배열의 역할\*\*

&#x20;  - f\_k = k·fs/N 을 계산해, FFT 결과의 각 인덱스 k(bin)가 실제 몇 Hz인지 알려주는 주파수 축 역할.

3\. \*\*0\~1초 vs 1\~2초 구간 주파수 차이\*\*

&#x20;  - 앞 1초는 440 Hz, 뒤 1초는 1000 Hz에 밝은 띠가 나타남. 시간이 지나며 강한 주파수가 440→1000 Hz로 바뀜.

4\. \*\*시간 그래프 vs Spectrogram, 무엇이 주파수 변화를 더 잘 보여주나?\*\*

&#x20;  - Spectrogram. 시간축과 주파수축을 동시에 보여줘 주파수가 바뀌는 순간을 한눈에 확인 가능.
